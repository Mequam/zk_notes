# sap/ewm/integration/itemCats

## About

items in delevery documents have categories, here
is a list of categories for your usage :)

- DLV : Standard Delivery Item
- PAC : Packing item
- RET : Returns Item
- TXT : Text Item
- VAL : Value Item (used only in outbound delivery processing)
